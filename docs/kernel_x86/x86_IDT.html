# IDT 中断描述符表

# IDT的作用

IDT和DGT一样是都是CPU的一个特殊的寄存器，IDT告知处理器在程序异常、硬件中断和软件中断发生时如何找到异常处理和中断处理程序，x86处理器的IDT可以容纳最多256个中断描述符。IDT由IDTR指定,执行汇编指令LIDT可以装入IDTR。

# IDTR

IDTR数据结构理论上可以存储在内存的任意位置，起始地址需要能够被8整除。
