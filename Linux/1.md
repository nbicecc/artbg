## Ubuntu apt 软件管理器国内软件源
### 更换软件源方法
```shell
# 备份
$ sudo cp /etc/apt/sources.list /etc/apt/sources.list.back
# 复制内容覆盖list文件中的内容
$ sudo vim /etc/apt/sources.list
# 更新
$ sudo apt-get update
# 如果出错恢复备份
$ sudo cp /etc/apt/sources.list.back /etc/apt/sources.list
```
